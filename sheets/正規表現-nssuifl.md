# 正規表現

## 基本

| | |
| --- | --- |
| . | 任意の1文字 |
| [ABC] | いずれか一文字 |
| [^ABC] | いずれかでない |
| (A|B|C) | A B Cのいずれか |
| ^ | 行頭 |
| $ | 行末 |

## 先読み・後読み

```
(?=pattern)
```

肯定先読み Lookahead

```
(?<=pattern)
```

肯定後読み Lookbehind

```
(?!pattern)
```

否定先読み Negative Lookahead

```
(?<!pattern)
```

否定後読み Negative Lookbehind


## サンプル

```
/^[a-zA-Z0-9.!#$%&'*+\/=?^_`{|}~-]+@[a-zA-Z0-9-]+(?:\.[a-zA-Z0-9-]+)*$/
```
