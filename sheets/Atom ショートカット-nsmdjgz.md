# Atomのショートカット

## 編集

| キー | 説明 |
| --- | --- |
| ctrl option b | コードの整形 |
| command / | コメント切り替え |
| command control ↑ | 上に行を移動 |
| command control ↓ | 下に行を移動 |

## 表示

| キー | 説明 |
| --- | --- |
| ctrl shift 9 | Git Tab 切り替え |
| ctrl shift 8 | Github Tab 切り替え |
| ctrl command p | Project Manager を開く |
| command \ | Tree View 切り替え |
