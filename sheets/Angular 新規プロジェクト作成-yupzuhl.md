# Angular 新規プロジェクト作成

```
ng new [PROJECT_NAME]
```


## オプション
 
| オプション | 説明 |
| --- | --- |
| `--skipInstall=true\|false` | trueの場合、パッケージのインストールをしない |
| `--style=css|scss|sass|less|styl` | cssの種類 |
| `--skipGit=true|false` | trueの場合、Gitリポジトリを作成しない |
| `--directory=directory` | インストールするディレクトリを指定する |
