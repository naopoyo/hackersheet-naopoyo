# Rails generate コマンド

## マイグレーション

```
rails g migration [MIGRATION_NAME]
```

## モデル

```
rails g model [MODEL_NAME]
```

## タスク

```
rails g task [TASK_NAME]
```

## ジョブ

```
rails g job [JOB_NAME]
```

## コントローラー

```
rails g controller [CONTROLLER_NAME]
```
