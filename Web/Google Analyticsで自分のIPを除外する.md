# Google Analyticsで自分のIPを除外する

1. 「**管理**」を開く
2. 「ビュー」の「**フィルタ**」を開く
3. 「**フィルタを追加**」

## フィルタを追加

1. ビューにフィルタを適用する方法を選択
   1. **新しいフィルタを作成** を選択
2. フィルタ情報
   1. **フィルタ名** を入力
   2. フィルタの種類
      1. フィルタの種類を選択: **除外** を選択
      2. 参照元かリンク先を選択します: **IP アドレスからのトラフィック** を選択
      3. 式を選択します: **等しい** を選択
      4. IP アドレスを入力

## 自分のIPを確認する

[確認くん](https://www.ugtop.com/spill.shtml)で自分のIPを確認。
