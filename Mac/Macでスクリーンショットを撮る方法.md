# Macでスクリーンショットを撮る方法

## スクリーンショットを撮る(画面全体)

- `⇧(Shift)` + `⌘(Command)` + `3`

## スクリーンショットを撮る(画面一部)

- `⇧(Shift)` + `⌘(Command)` + `4`

## 画面を録画

- `⇧(Shift)` + `⌘(Command)` + `5`
- 停止する場合
  - `⇧(Shift)` + `⌘(Command)` + `5` でメニューを出して停止ボタンを押す。
  - メニューバーにある停止ボタンを押す。
- オプションで保存先を変更可能。
