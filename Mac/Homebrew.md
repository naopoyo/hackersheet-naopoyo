# Homebrewのはじめかた

## インストール

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

.zshrc

```
export PATH="/opt/homebrew/bin:$PATH"
```

## アップデート

```
brew update
```


## コマンド

```
brew doctor
```

```
brew update
```

```
brew list
```
