# Markdown記法

## 見出し

```markdown
h1
========
```

```markdown
h2
————
```

```markdown
# h1
```

```markdown
## h2
```

```markdown
### h3
```

```markdown
#### h4
```

```markdown
##### h5
```

```markdown
###### h6
```


## リスト

```markdown
- Item1
- Item2
  - Item2-1
  - Item2-2
- Item3
```

```markdown
1. Item1
2. Item2
3. Item3
```


## コード

### インライン

```markdown
`some code`
```

### ブロック

````markdown
```javascript
(() => {
  'use strict';

  console.log('Hello world');
})();
```
````

## 強調

```markdown
*強調*
```

```markdown
**強調**
```

```markdown
_強調_
```

```markdown
__強調__
```


## リンク

```markdown
[Title](https://example.com)
```
