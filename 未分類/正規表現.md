# 正規表現

## 基本

| 正規表現          | 対象            |
| ----------------- | --------------- |
| .                 | 任意の1文字     |
| [ABC]             | いずれか一文字  |
| [^ABC]            | いずれかでない  |
| (A&#124;B&#124;C) | A B Cのいずれか |
| ^                 | 行頭            |
| $                 | 行末            |

## 先読み・後読み

### 肯定先読み Lookahead

```
(?=pattern)
```

### 肯定後読み Lookbehind

```
(?<=pattern)
```

### 否定先読み Negative Lookahead

```
(?!pattern)
```

### 否定後読み Negative Lookbehind

```
(?<!pattern)
```

## サンプル

### メールアドレス

```
/^[a-zA-Z0-9.!#$%&'*+\/=?^_`{|}~-]+@[a-zA-Z0-9-]+(?:\.[a-zA-Z0-9-]+)*$/
```

### IPアドレス

```
/(([1-9]?[0-9]|1[0-9]{2}|2[0-4][0-9]|25[0-5])\.){3}([1-9]?[0-9]|1[0-9]{2}|2[0-4][0-9]|25[0-5])/
```
