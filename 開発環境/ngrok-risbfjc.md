# ngrok

ローカル環境を外部公開できるサービス。

## 公式

[ngrok - secure introspectable tunnels to localhost](https://ngrok.com/)


## はじめに

公式サイトでアカウント作成する。

## インストール

```
brew install --cask ngrok
```


## 設定ファイルのパス

~/.ngrok2/ngrok.yml

## 使い方

```
ngrok http 80
```
