# anyenvとrbenvでRuby環境構築

```
anyenv install rbenv
```

インストール

```
exec $SHELL -l
```

シェルを再起動

```
rbenv install -l
```

インストール可能なバージョンを確認

```
rbenv install [VERSION]
```

```
rbenv global [VERSION]
```

グローバル環境に設定

```
rbenv rehash
```
