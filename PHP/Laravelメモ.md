# Laravelメモ

## プロジェクト作成

```
composer create-project laravel/laravel laratest
```

## ビルトインサーバ起動

```
php artisan serve --host 0.0.0.0 --port=8080
```

## composerでphp-cs-fixerをインストール

```
composer global require friendsofphp/php-cs-fixer
```
